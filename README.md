# ☕ 杭咖 HANG COFFEE - 城市连锁咖啡品牌智慧运营仿真平台

> **"温暖的商业智能"** —— 将专业的数字化商业分析与温馨的咖啡文化深度融合，为城市连锁咖啡品牌打造的 100% 纯前端、高性能数字化运营决策支持系统。

---

## 🌟 平台简介

**杭咖 HANG COFFEE** 是一款专为咖啡连锁品牌量身定制的智慧运营与市场仿真决策平台。平台通过先进的前端微观仿真引擎，模拟真实的市场竞争、多维的顾客行为以及复杂多变的运营场景。管理者可以在地图上选址开店、调整成本参数、优化产品定价与库存，并通过实时同步的数据大屏直观感知策略成效，实现“数据驱动、精准决策”。

---

## 🎨 核心设计理念

平台遵循 **"温暖的商业智能"** 哲学，在保障专业数据分析的前提下，创造一个具有亲和力的数字化环境。

*   **色调体系**：采用精心调配的咖啡色系搭配温润的米色，拒绝冰冷的高对比度色彩。
    *   🟫 **深咖啡棕 (`#3C2415`)**：品牌主色，彰显专业、稳重的商业质感。
    *   🪵 **浅咖啡棕 (`#8B5A2B`)**：辅助色，用于精细的次要元素及边框描边。
    *   ✨ **咖啡金 (`#D4A574`)**：强调色，凸显关键KPI数据与高亮交互。
    *   🍦 **米白色 (`#FAF7F2`)**：系统背景，营造温暖舒适的视觉包容度。
*   **字体系统**：主标题选用优雅的思源宋体 (*Noto Serif SC*) 与品牌英文衬线体 (*Oranienbaum*)，正文搭配思源黑体 (*Noto Sans SC*)，保证可读性的同时传递古典优雅的质感。
*   **精美动效**：基于 **Anime.js** 打造 KPI 数据滚动、卡片浮动、图表平滑绘制及柔和的粒子背景，为严肃的商业决策增添呼吸感。

## 🛠️ 核心功能模块

平台由四个相互协同的核心系统组成，构建了完整的咖啡连锁品牌智慧运营生态：

```
                    ┌─────────────────────────┐
                    │     基础数据配置面板      │
                    │ (店铺/客户/产品数据构建)  │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │     智能仿真控制面板      │
                    │ (时间轴/天气/策略动态调节) │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │    数据可视化决策大屏     │
                    │ (KPI仪表/地图热力图/分析)  │
                    └────────────┬────────────┘
                                 │
                                 ▼
                    ┌─────────────────────────┐
                    │    策略实验与报告系统    │
                    │ (A/B测试/数据一键导出)     │
                    └─────────────────────────┘
```

### 1. 📋 基础数据配置面板 (`management.html`)
*   **地图选址系统**：集成交互式杭州地图，点击候选区域即可定位，配置旗舰店、快取店或社区店三种店铺类型。
*   **参数动态滑块**：可视化调节租金、人员配置、薪水等级及水电成本，系统自动预估月度运营成本。
*   **客户画像细分**：提供商务白领、在校学生、游客、社区居民四种画像，支持微调其价格敏感度、品质偏好与购买频次。
*   **产品菜单管理**：支持对咖啡、茶饮、烘焙点心进行售价与原材料成本配置，可设定库存预警阈值。

### 2. ⚡ 智能仿真控制面板 (`simulation.html`)
*   **时间控制器**：支持 1x 至 100x 速度调节的时间轴，可自由控制仿真的启动、暂停与重置。
*   **环境场景触发**：内置晴天、雨天、雪天及高温天气，并能一键开启第二杯半价、会员日、新品推广等营销活动。
*   **竞争对手模拟**：内置瑞幸、星巴克等周边竞品促销活动对客流的实时分流影响，考验策略韧性。

### 3. 📊 数据可视化决策大屏 (`dashboard.html`)
*   **热力分布地图**：结合杭州地理热力图，实时展示各门店订单量、销售额及客流量分布。
*   **核心 KPI 看板**：实时滚动更新总销售额、订单量、客单价等核心指标，畅销产品与明星门店动态排行。
*   **多维度分析面板**：提供 24 小时客流曲线图、产品关联购买（套餐推荐）分析及会员复购率走势。

### 4. 📝 策略实验与报告系统 (`reports.html`)
*   **A/B 测试对比**：支持同时运行两套仿真环境，并排对比不同定价与营销策略下的收益回报。
*   **一键导出数据**：支持将订单明细、经营流水与运营指标导出为 Excel 或 JSON 格式，方便二次分析。

## 🚀 快速上手与使用

由于平台采用了**纯前端高性能架构**，无需配置复杂的服务器环境，即可在本地秒级启动。

### 📖 使用步骤：
1.  **克隆或解压**本项目的源文件包。
2.  在现代浏览器（推荐 Google Chrome, Microsoft Edge）中直接**双击打开 `index.html`**。
3.  点击首页的 **"立即开始仿真"** 或通过顶部导航栏进入各核心子模块。
4.  在数据管理中初始化数据后，进入仿真引擎页面，点击“启动仿真”，即可查看实时模拟运作！

## 💻 技术架构与选型

系统追求轻量与高效的平衡，精心选用主流前端依赖，保证了离线状态下的强劲表现：

*   **核心逻辑与状态**：`Vanilla JavaScript (ES6+)` 驱动底层仿真引擎与数据统计，实现毫秒级反应与数据完全本地化保护。
*   **界面样式**：使用 `Tailwind CSS` 进行极速、现代的响应式界面构建，辅以自定义 CSS 实现温暖的毛玻璃、卡片悬浮与咖啡微动效。
*   **数据可视化**：采用 `ECharts.js` 绘制丝滑过渡的多维度折线图、柱状图和雷达图。
*   **地理信息交互**：基于 `Leaflet.js` 渲染轻量、高性能的杭州市地图，承载店址分布与经营热力。
*   **交互动画**：使用 `Anime.js` 库，管理数字滚动、卡片入场和动态组件变化。
